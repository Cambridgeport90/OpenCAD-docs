OpenCAD Installation Requirements include the following:

  • Operating System: Linux or Windows
  • Webserver: Apache or Nginx
  • PHP: 7.3 (Recommended), though 7.2 is the minimum
  • Database: MySQL ≥ 5.5 or MariaDB ≥ 10.0, though we recommend 10.3

 

Are you ready to get started with OpenCAD? Follow our Installation Guide!

 

NOTE: We do NOT support “free” web hosts or web host services because of instability and inconsistencies.
